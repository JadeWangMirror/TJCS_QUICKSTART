# TJCS QUICKSTART

同济大学计算机科学与技术专业课程资料整理仓库。


## 快速开始

部分大文件（教材/笔记/工具包）超过 GitHub 单文件限制，已用 Git LFS 管理，克隆前建议先启用：

```bash
git lfs install
git clone https://github.com/JadeWangMirror/TJCS_QUICKSTART.git
```

## 参考作业

学生作业成果不入仓，以外部链接形式维护：
- 跨课程的「集中式作业大仓库」：见 [reference.md](./reference.md)
- 单门课程的作业链接：见各课程目录下的 `reference.md`

## 整理规范

按「学期 / 课程名」组织。每门课程目录下有课程级说明 `<课程名>.md`（多学期、多老师共用）和 `reference.md`（参考作业链接）；各老师资料放在 `学期编码-教师缩写/` 子目录下，按 `ppt/`、`notes/`、`exam/`、`tasks/` 等分类。教材不入仓（只在 md 写书名），学生作业源码不入仓（只维护外部链接）。**整理规范见 [AGENT.md](./AGENT.md)（整理协作规范）。**
