# 期末考试与复习

## 重点定理与证明思路

- p29 划分超限性

  第二类 Catalan 意义运用，证明使用生成函数类似的方法，对 `x` 求 `n` 次导。

- p31 二项式系数界值定理

  在题目范围内证明 `log` 二项分布的概率相乘项大于 `-n` 信息熵，放缩求解。

- p41 丢番图逼近

  `0-1` 切分成 `n` 段，对于 `kx` 的小数部分，`k` 为 `0` 到 `n`，必定有两个 `k` 使得两个小数部分在一个区间，换元整数部分，凑出小数部分不等式，再换元得证。

- p43 斐波那契余数列周期性

  考虑相邻的余数有 `m * m` 种可能，但相邻余数相加的余数只有 `m` 种可能，所以一定存在相邻余数组成相同的情况，即周期。

- p45 单调子数列

  限制 `ak` 开始最长子数列长度小于其中一个条件，鸽巢一般形式知至少有部分相等，取出这一部分。假设开始设的递增，则其角标越小首元素越大，否则就可以该首元素加一个一样长的后续子数列破坏预设，这些首元素满足另一个条件，反之亦然。

- p58 双色 Ramsey 三角不等式

  假设 `n` 为双色 Ramsey 的右侧，若证明其可以取出 `ka`、`kb` 即得证。对一个点分析，完全图所以连出去 `n - 1` 条边，列表可得红蓝边数量关系，总能满足不等式右侧的某个 Ramsey 数。对于连出来的这些点，肯定满足 `ka`、`kb` 中的一个，不满足的那个 `ki` 算上分析点即成立。

- p60 对角双色 Ramsey 范围

  非对角 Ramsey 上界用数学归纳和三角不等式。对角 Ramsey 的上界沿用非对角的带入放缩可得。主要关注下界：设 `n` 是所求下界，`Kn` 有 `n * (n - 1) / 2` 条边，因此有 `2` 的这么多次幂种染色方案。若对角双色 Ramsey 比 `n` 小，则 `Kn` 不论怎么染色都会有同色 `Ka`，每个染色方案都有。但是对于一个 `Ka` 来说，在 `Kn` 中充其量有 `2^(n * (n - 1) / 2 - a * (a - 1) / 2)` 个，也就是除了 `Ka` 的边以外随便染色，可以有 `C(n, a)` 个这样的 `Ka`，这个总数乘以 `2`，即 `Ka` 本身可以是全红或者全蓝，比总染色方案小，由此反证。

- p66 舒尔数（有和子集）

  本质上只要证明 `n` 维的 `R(3, 3, ...)`，即不管怎么取都有三角形，是一个舒尔数就可以。规定一种染色方案：顶点编号差属于划分出的某集合，这两个顶点的连边就用该集合编号染色。一共划分 `n` 个集合，也就是 `n` 种颜色。由于 Ramsey 的性质，会有一个同色三角形，三个顶点的差值都属于一个集合，换元可得其有和关系，得证。

- p79 生成函数卷积证明范德蒙恒等式

  很优雅的证明，而且辅助记忆两个公式。卷积和范德蒙都是角标和是 `k`，用 `(1 + x)^m` 和 `(1 + x)^n` 进行卷积运算，再用 `(1 + x)^(m + n)` 的生成函数。

- p99 齐肯多夫定理

  定理形式可以表示为任意正整数都可以拆解为一组不连续斐波那契数列项的和，且拆解方式唯一。用第二数学归纳，小于 `n` 的所有数都可以这么拆。对于这个 `n` 来说，它肯定位于两个连续斐波那契数列中间，不妨设为第 `k` 和 `k + 1` 项的 `F`。`n - Fk` 显然小于 `n`，可以唯一拆；同时 `n - Fk` 放缩可知小于 `F(k - 1)`。设 `n - Fk` 中拆出的最大项是 `Fk*`，则这一项小于 `n - Fk`，也小于 `F(k - 1)`，所以和 `Fk` 不相邻，移项得证。

## 高频复习内容

- Catalan：走方格、出入栈、三角剖分数。
- 两类 Stirling 数的含义与推导：第一类切 `k` 个圆排，第二类切 `k` 个非空子集。
- 简单一般形式的容斥原理：错排、依赖变量个数。这两个都是对定理的运用，建议看完加深理解，尤其是对小 `omega m` 项含义的理解，即有 `m` 个满足即可，可以满足更多。
- 棋子多项式、二重错排：建议重点掌握，至少背下方法。

## 莫比乌斯反演与欧拉函数

- p132 莫比乌斯反演

  其一，`n` 的所有因子的莫比乌斯函数函数值之和为 `0`，在 `n = 1` 的时候为 `1`。证明技巧：因为莫比乌斯函数会自动忽略非一阶的素因子的影响，可以直接将 `n` 素因子化为 `n*` 处理。一共 `m` 个因子，选 `k` 个计算，再对 `k` 从 `1` 到 `m` 遍历可求得为 `0`。

  其二，`d` 为因子，`d*` 为 `d` 的因子，对这样的 `d / d*` 的莫比乌斯函数值求和，为 `0`，`n` 是 `d*` 时为 `1`。

  有个数论引理：一个数和它的因子的因子的商，是这个数和它的因子的商的倍数。

  先说明这个引理如果成立，换元即可得证。

  其三，莫比乌斯反演公式：

  ```text
  f(n) = n 的所有因子在 g 上的值的求和
  g(n) = n 的所有因子的莫比乌斯函数值 * f(n / d) 的求和
  ```

  证明暂时没找出技巧，有一个和式变换的求和顺序改变技巧，建议看书。
