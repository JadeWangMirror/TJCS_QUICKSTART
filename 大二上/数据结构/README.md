## 期末考试与复习

期末一般有选填和大题，选填一般刷熟408王道数据结构分册即可，大题主要有1.给定某一算法，要求给出算法执行过程中的中间状态表示，例如堆栈和排序等，参考王道即可 2.设计算法和伪代码，难度较低。
复习时主要参考王道，也可以看课程ppt。
