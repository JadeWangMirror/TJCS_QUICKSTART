# 参与贡献

欢迎补充作业参考、课程资料或课程体验。两种常见贡献方式：

## 1. 补充自己的作业

作业源码/文件不入仓，只放外部链接：

- **单门课程**的作业 → 加到对应课程目录下的 `reference.md`（按作业类型分节，注明昵称 + 仓库链接 + 简介）。
- **多门课程集中在一个大仓库** → 加到根目录的 [`reference.md`](./reference.md)。

## 2. 补充或更新课程资料

- **新增某位老师独有的资料**（课件、笔记、试卷等）：在课程目录下建 `学期编码-教师缩写/` 文件夹（如 `25262-wzh/`），资料按 `ppt/`、`notes/`、`exam/`、`tasks/` 等分类放入。**尽量只加自己特有的文件，避免和已有资料重复。**
- **仅更新课程说明**（课程体验、给分、考勤、复习建议等）：直接编辑该课程的 `<课程名>.md` 即可，不需要建老师文件夹。

> 完整的目录结构、命名规则与整理流程见 [AGENT.md](./AGENT.md)。
