# 课程表管理系统操作文档

## 项目概述

本系统是一个集成了AI课程推荐功能的课程表管理系统，包含用户注册登录、课程选择、课表保存等功能。

## 系统架构

- 前端：Vue3 + TypeScript + Vite
- 后端：Flask + SQLite
- 数据库：SQLite

## 安装与启动

### 1. 环境准备

确保已安装以下软件：
- Python 3.8+
- Node.js 16+
- pnpm (推荐)

### 2. 后端设置

1. 进入后端目录：
   ```bash
   cd backend
   ```

2. 安装Python依赖：
   ```bash
   pip install -r requirements.txt
   ```
   
   如果没有requirements.txt文件，需要安装以下包：
   ```bash
   pip install flask flask-cors pandas sentence-transformers torch werkzeug jwt
   ```

3. 启动后端服务：
   ```bash
   python app.py
   ```
   
   服务将运行在 http://localhost:5000

### 3. 前端设置

1. 进入前端目录：
   ```bash
   cd frontend
   ```

2. 安装依赖：
   ```bash
   pnpm install
   ```
   
   或者使用npm：
   ```bash
   npm install
   ```

3. 启动前端开发服务器：
   ```bash
   pnpm dev
   ```
   
   服务将运行在 http://localhost:5173

## 功能说明

### 1. 用户注册与登录

#### 注册流程
1. 访问系统，点击"立即注册"链接
2. 输入姓名、邮箱和密码
3. 点击"注册"按钮完成注册

#### 登录流程
1. 访问系统，输入已注册的邮箱和密码
2. 点击"登录"按钮即可进入系统

### 2. 课程选择

1. 登录后，在左侧课程列表中可以看到所有课程
2. 点击"添加"按钮将课程加入选课列表
3. 在课表视图中，可以点击课程块右上角的"X"按钮移除课程

### 3. AI课程推荐

1. 在左侧的AI搜索框中输入您想学习的课程描述
2. 点击"搜索"按钮，系统将推荐相关课程
3. 点击"清空"按钮可恢复显示所有课程

### 4. 课表管理

#### 保存课表
1. 在用户主页区域，找到"保存当前课表"部分
2. 输入课表名称和描述
3. 点击"保存课表"按钮

#### 查看与加载课表
1. 在用户主页的"我的课表"部分可以看到已保存的课表列表
2. 点击"加载"按钮可以将该课表加载到当前视图
3. 点击"删除"按钮可以删除该课表

## 数据库结构

系统使用SQLite数据库，包含以下表：

### users 表
- id: INTEGER PRIMARY KEY - 用户ID
- email: VARCHAR(255) - 用户邮箱（唯一）
- password_hash: VARCHAR(255) - 密码哈希值
- full_name: VARCHAR(255) - 用户姓名
- avatar_url: VARCHAR(500) - 头像URL
- created_at: DATETIME - 创建时间

### timetables 表
- id: INTEGER PRIMARY KEY - 课表ID
- user_id: INTEGER - 所属用户ID（外键关联users表）
- name: VARCHAR(255) - 课表名称
- description: TEXT - 课表描述
- courses: TEXT - 课程数据（JSON格式）
- created_at: DATETIME - 创建时间

## API接口

### 用户相关
- `POST /api/register` - 用户注册
- `POST /api/login` - 用户登录

### 课表相关
- `GET /api/timetables` - 获取用户课表列表
- `POST /api/timetables` - 保存新课表
- `GET /api/timetables/:id` - 获取指定课表
- `PUT /api/timetables/:id` - 更新指定课表
- `DELETE /api/timetables/:id` - 删除指定课表

### 课程相关
- `GET /api/courses` - 获取所有课程
- `POST /api/ai-search` - AI课程搜索

## 安全说明

1. 密码使用哈希算法存储，不会明文保存
2. 用户认证使用JWT token，有效期限为30天
3. 所有API请求需要认证token才能访问用户数据

## 常见问题

1. **无法连接后端服务**
   - 检查后端是否已启动（http://localhost:5000）
   - 检查防火墙设置

2. **AI搜索功能不工作**
   - 确保网络连接正常
   - 检查后端日志是否有错误信息

3. **登录失败**
   - 检查邮箱和密码是否正确
   - 确认已成功注册账户
