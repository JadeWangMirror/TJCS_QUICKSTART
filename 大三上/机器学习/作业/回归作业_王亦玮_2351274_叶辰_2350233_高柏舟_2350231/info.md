#### **B. 模型架构与优化技术**

* **线性正则化家族**：
* 
**Ridge (L2)**：增加权重平方和惩罚项，有效降低了模型的方差 。


* 
**Lasso (L1)**：通过权重绝对值惩罚，实现了特征的稀疏化选择 。


* 
**Elastic Net**：平衡了 L1 的稀疏性和 L2 的稳定性 。




* 
**决策树回归 (Non-linear)**：使用 **均方误差 (MSE) 减少量** 作为分裂标准，递归构建树结构，捕捉数据中的非线性交互关系 。


* 
**训练策略**：线性模型采用了 **小批量随机梯度下降 (Mini-batch SGD)** ；决策树采用了 **渐进式训练监控** 与 **早停 (Early Stopping)** 策略来控制过拟合 。

### 4. 优化空间建议

基于你目前的工作，可以从以下几个方面进一步精进：

1. **特征交互工程 (Feature Interaction)**：
* 报告中提到了 BMI 与吸烟状态的组合效应 。你可以在线性模型中手动添加这些二阶特征（如 `bmi * smoker_yes`），这可能会显著提升线性模型的 R² 表现。




2. **集成学习 (Ensemble Learning)**：
* 既然单棵决策树表现优异 ，引入 **随机森林 (Random Forest)** 或 **梯度提升树 (GBDT/XGBoost)** 将能进一步降低单棵树的高方差问题，获得更平滑、更稳定的预测曲线。




3. **学习率调度 (Learning Rate Scheduler)**：
* 代码中线性回归的学习率 `lr=1` 较高且固定，导致训练过程出现剧烈波动 。建议采用 **学习率衰减策略**（如余弦退火），使模型在训练后期能够平稳收敛到更优的局部最小值 。

* 目前使用手动设置的 Lambda 候选值 。可以考虑使用 **贝叶斯优化** 代替网格搜索，在更广阔的参数空间内更高效地寻找最优的正则化强度。
