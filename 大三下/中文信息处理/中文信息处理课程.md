# 中文信息处理 · 课程资料

本课程的核心实验是一个**中文分词系统对比实验**，完整代码与报告放在独立仓库：

🔗 <https://github.com/hyxtj/nlp-cws-methods>

## 仓库简介

基于 **SIGHAN Bakeoff 2005 PKU** 数据集，实现并对比四种中文分词方法：

| 方法 | 模型 |
| --- | --- |
| A | 词典最大匹配（FMM / BMM / BiMM） |
| B | 隐马尔可夫模型 HMM（BIES + Viterbi） |
| C | BiLSTM-CRF |
| D | BERT 微调（bert-base-chinese） |

## 你能找到什么

- 四种分词方法的完整实现（`method_a` ~ `method_d`）
- 训练 / 评估 / 对比脚本（`train_abc.py`、`train_bert.py`、`evaluate.py`、`compare_all.py`）
- 交互式分词演示（`main.py`）
- 数据集（PKU / MSR）与实验报告（`实验报告.md`）

> 全部可在 CPU 上运行，普通笔记本即可复现。
